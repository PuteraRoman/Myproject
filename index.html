HTML - tags

Header tags

"h1" tag - used for main headings.

"h2" tag - used for subtitles.

"h3, h4, h5" tags - used to denote different levels of headers.

Descriptive tags

"main" tag - helps other developers find the main content of your page.

"header" tag - represents introductory content, usually a group of introductory or navigational aids.

"footer" tag - represents a footer for its nearest sectional content or sectional root.

"article" tag - represents an independent part of a document, page, or site intended for independent distribution or reuse.

Element "img"

"img" tag - embeds an image in a document.

Unordered lists

Unordered lists begin with an opening "ul" element, followed by any number of "li" elements. Unordered lists are closed "/ul".

Example:

  • Example_1
  • Example_2
  • Example_3

Ordered lists

Ordered lists start with an "ol" element followed by any number of "li" elements. Finally, ordered lists are closed with the "/ol" tag.
